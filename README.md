# Система управления автономными грузовыми роботами

**Веб-приложение для удалённого управления множеством автономных роботов-грузоперевозчиков с интеллектуальным планированием маршрутов.**

Этот проект представляет собой  систему, позволяющую пользователям подключаться к автономным роботам, строить оптимальные траектории перемещения с помощью OCP-планировщика (оптимального управления) и управлять выполнением заданий через интуитивный веб-интерфейс.

## Архитектура

Система построена по классической трёхзвенной архитектуре:

- **Frontend**: React-приложение с интерактивной картой для визуализации роботов и построения маршрутов
- **Backend**: FastAPI REST API, обрабатывающий бизнес-логику и OCP-планировщик
- **Database**: MariaDB с моделями роботов, карт, пользователей, вычисленных траекторий и транспортных заявок
- **Infrastructure**: Nginx как reverse proxy, развертывание на виртуальной машине Debian

## Ключевые возможности

- **Управление роботами**: Просмотр статуса, уровня заряда и местоположения всех роботов в реальном времени
- **Интеллектуальное планирование**: Интеграция с OCP-планировщиком для построения оптимальных траекторий с учётом динамики робота-велосипеда
- **Интерактивный интерфейс**: Визуальное задание целевых точек на карте, просмотр рассчитанных маршрутов

## Технологический стек

- **Backend**: Python, FastAPI, SQLAlchemy, Uvicorn
- **Frontend**: React, Axios для API-запросов
- **Database**: MariaDB
- **DevOps**: Nginx, bash-скриптинг, виртуальные машины Debian
- **Тестирование**: Locust для нагрузочного тестирования
